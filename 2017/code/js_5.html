JS libraries

You can import JS libraries and use pre-written functionalities

That allows us to do not need to reinvent the wheel and start coding on top of what other people have already written

For example, we can import a library that already has functionality to generate a sine wave and use it at will

From within the HTML code, we can load the library by providing a local link or an Internet link. For example, we can load the Pizzicato JS library directly from the web:

<script src=https://cdnjs.cloudflare.com/ajax/libs/pizzicato/0.6.1/Pizzicato.min.js>

Then we can instantiate a sine wave on the browser by just doing
var sineWave = new Pizzicato.Sound({
source: 'wave',
options: {
frequency: 440
}
});

sineWave.play();

In-class assignment

Using the Pizzicato Javascript library, create a web-based theremin

The theremin should be controled by the position of the mouse within a zone of the screen (you can use a div for this). The theremin should generate sound only when the mouse is inside this zone.

Horizontal axis will control the frequency of the theremin, and the vertical axis the level

Steps:

  1. Create a container box (using a div and a css for setting the size)
  2. Create two functions: one that will be called when the mouse is inside the div and another one that will be called when the mouse is out of the div. You may want to use the events 'onmousemove' and 'onmouseleave'
  3. Add functionality to get mouse position from within the playback zone. Check the mouse events: event.clientX and event.clientY
  4. Load the Pizzicato JS library in your code
  5. Instantiate the sineWave
  6. Control the sound level and frequency with the mouse position. Check the ranges.
  7. Add the ability to stop the sound if mouse is outside of the playback zone
  8. Add a delay
