Forms and inputs

HTML forms are used to collect user input. HTML forms can contain input elements like text fields, checkboxes, radio-buttons, submit buttons and more.

Forms' syntax uses the form tag to create an HTML form, then

  • the input element is used to select user information,
  • the type attribute to specify the type of data, and
  • the name attribute stores the user-entered data

A simple example of an input text field would be:

        <form>
            <p>
                User name:
                <input type="text" name="userName">
            </p>
            <p>
                Artist name:
                <input type="text" name="artistName">
            </p>
        </form>

Using the data from forms in JavaScript

We usually want to use the data a user entered into a form. In order to accomplish this, we create a form and attach it to the DOM by means of assigning and ID to the form element: 
<form id="userdata">
	User name: <input type="text" name="userName">
	<input type="radio" name="sex" value="male"> Male
	<input type="radio" name="sex" value="female"> Female
        <br>
	Password: <input type="password" name="pwd">
        <br>
	Artist name: 
        <input type="text" name="artistName">
        <br>
	<input type="checkbox" name="retrieve" value="discography">Retrieve album names<br>
	<input type="checkbox" name="retrieve" value="members">Retrieve member names<br>
</form>

Then, we can access the data within the form. For example, we can add a button that, when clicked, calls a function: 

        <input type="button" onclick="queryData()" value="Retrieve">

Once the function is called, we can retrieve the data from the form that is attached to the DOM:

        function queryData() {
            var x = document.getElementById("userdata");
            userName = x.elements["userName"].value;
            artistName = x.elements["artistName"].value;          
        }

Then, we finally can do something with that data. For example, we use the values entered by the user for userName and artistName for creating a request to another server, or we attach these values to the DOM and display them in the page.

In-class assignment

Create a form for asking the name of a user and the name of an artist for which the user is interested about, or for which is looking for information.

When the user press a button, the name of the user and artist should be printed on the screen.

Steps

  1. Create the webpage skeleton
  2. Create the form
  3. Create a button that triggers a function when clicked
  4. Create the function
  5. The function should read the elements from the form and store the name of the user and the name of the artist in two variables
  6. Print the user and artist names into the console
  7. Create an HTML element that will act as a placeholder for rendering the user and artist names
  8. From within the function, populate the placeholder element
