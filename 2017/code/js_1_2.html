MUMT301

In-Class exercise: implement a web-based Theremin.

For the synthesis engine, you may use the Pizzicato Javascript library.

How many gestural variables has a Theremin?

Steps for the assignment:
  • Code the basic HTML code for the Theremin, including loading the Pizzicato Library
  • Create a container box that will be the performance space (use div elements and)
  • Add functionality to get mouse position from within the performance zone
  • Instantiate the sineWave
  • Control the sound level and frequency with the mouse position
  • Add the ability to stop the sound if mouse is outside of the playback zone
  • Add a delay
