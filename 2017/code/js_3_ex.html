Music APIs

In-class exercise

Let the user enter the name of a song. The system should return a list of existing recordings in the the Musicbrainz API for that song.

Steps:

  1. Create form to retrieve user's data
  2. Parse the data from the form to extract the recording's name
  3. Determine the MusicBrainz call you have to make to retrieve the list of recordings
  4. Make the HTTP request call. You can make use of the httpGet function in this page
  5. Parse the returned array
  6. Render the data into the Web browser (using divs would be nice)
  7. Style the results
