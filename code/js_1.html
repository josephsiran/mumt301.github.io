    The <script> tag

    The lines between <script> and </script> contain the JavaScript code, and the browser will interpret the code between them as JavaScript.

    Each line of JavaScript ends with a semicolon

    Scripts can be placed in any section of HTML code, however putting scripts at the bottom of the <body> section of a web page can reduce display time.

    Anyway, separating HTML and JavaScript by putting all the code in one place, is always a good habit.

    As the CSS files, scripts can also be placed in external files. This is practical when reusing the same code.

    JavaScript files have the file extension .js. To use an external script, put the name of the script file in the source (src) attribute of the <script> tag:

    <script src="myScript.js"></script>

    JavaScript Syntax

    In JavaScript there are variables which are names to store data values.

    var x, length;
    x = 5;
    length = 6;

    or

    var x = 5;
    var length = 6;

    Also, arithmetics operations can be performed suing JavaScript variables, as in:
    var y = 5;
    var x = y + 2;

    JavaScript datatypes

    Strings

    var artistName = "Ratatat ";
    var albumName = "Classics";
    var artistalbumName = artistName + albumName;

    or

    var artistalbumName = artistName.concat(albumName)

    Numbers

    var albumPrice = 17.99;
    var songNumber = 8;

    Booleans

    var listenerHas = true;
    var downloaded = false;

    Arrays

    An array is a special variable that can hold more than one value at a time.

    Arrays are written in brackets, separated by commas

    var albums = ["Ratatat", "Classics", "LP3", "LP4"]

    You refer to an array element by referring to the index number.

    albums[4] = "Magnifique"

    You can extract some characteristics of the array by looking at its properties. For example, to know the current length of the array:

    albums.length

    Objects

    JavaScript variables are containers for data values.

    Objects are variables too, but objects can contain many values.

    Objects are written with curly braces, properties are written as name:value pairs separated by commas

    var release = { albumName:"Classics", artistName:"Ratatat", year:2006, label:"XL Recordings" }

    Objects are data with added properties (i.e., values associated with objects) and methods (i.e., actions objects can perform)

    Example: an album release object

    release.albumName = "Classics"
    release.artistName = "Ratatat"
    release.year = "2006"
    release.label = "XL Recordings"
    release.like()
    release.addPlaylist()
    release.download()
    release.rate()

    New attributes can be easily added

    release['members'] = "Evan Mast, Mike Stroud"

    Accesing object properties

    release.albumName;
    release["albumName"];

    JavaScript Events


    JavaScript can be used to execute code when events are detected

    These events are actions that happen to HTML elements. For example:

    • An HTML element finished loading
      • <element onload="myScript()">
    • An HTML input field was changed
      • <select onchange="myFunction()">
    • An HTML button was clicked
      • <button type="button" onclick="otherScript()">

    The Document Object Model

    • The Document Object Model (DOM) is a platform and language-neutral interface that allows programs and scripts to dynamically access and update the content, structure, and style of a document

    • When a web page is loaded, the browser creates a Document Object Model of the page

    • The DOM is constructed as a tree of objects


    • With the object model, JavaScript gets all the power it needs to create dynamic HTML. By doing so, we can select elements from the DOM, change their attributes, and append new elements to it
