Recap

We use

  • HTML to define the content of web pages
  • CSS to specify the layout of web pages
  • JavaScript to program the behavior of web pages

    • And so, if we want to embed a media player to play back an audio clip, we create an HTML element (only supporting MP3, ogg, and WAV audiofiles) as: 

    <audio> id="this_player" controls> 
	<source src="./link/to/audiofile"> 
</audio>

    The <audio> tag defines sound content, and the controls attribute adds audio controls, like play, pause, and volume.

    However, we can also change the source of <audio> by calling a script to change its source. For example, we can define two functions, one for each sound: 
    <script> 
	function audio1 () {
		document.getElementById("this_player").src = "./sounds/uncompressed/handrum2.wav";
		}
	function audio2 () {
		document.getElementById("this_player").src = "./sounds/uncompressed/s1.wav";
		}
</script>
     These functions have to be called from an action performed somewhere by someone (e.g., by the user, by the browser, at a specific time, ...) So far, we have learned how to perform actions with buttons. And so, we can call the functions audio1 or audio2 when a button is pressed, as in 
    <button> onclick=audio1()> audio1 </button>
<button> onclick=audio2()> audio2 </button>

    A better way of doing it...

    However, usually in computer programming when you are coding the same thing multiple times you are doing something wrong. In the previous example, both functions audio1 and audio2 share the same functionality, and so the only thing that changes is the audiofile link. And so, in this case, only one function is needed, and we will pass the link as an argument to the function. 

    <script> 
	function audioLink (link) {
		document.getElementById("new_player").src = link;
		}
</script>
     And so, we create a new <audio> element 
    <audio id="new_player" controls> </audio>

    and also we create buttons whose actions on them point to the audioLink function, and we pass the individual links as arguments to this function: 
    <button onclick=audioLink("./sounds/uncompressed/handrum2.wav")> audio1 </button>
<button onclick=audioLink("./sounds/uncompressed/s1.wav")> audio2 </button>
