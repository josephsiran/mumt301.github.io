JavaScript Intro

When you access a website, the website is very likely made out of three components

  • HTML code, used to indicate the content of the website.
  • CSS code, used to indicate the aesthetics (and layout) of website.
  • JavaScript code, used to control (or alter) the behavior of the website.

JavaScript is the programming language of the web All modern web browsers support it

JavaScript can change the content and attributes of HTML elements:

Running JavaScript code. Time to fire up your JavaScript interpreter.

  • You can (luckily) access a JavaScript interpreter from your favorite web browser.
  • The tools offered by modern browsers allow you to debug (finding and correcting errors) the JavaScript code of your website.
  • You can achieve this by using the JavaScript console (interpreter), reading the HTML source, and CSS attributes.
  • One common way of doing it is using the console.log() from your JavaScript code to display values and messages in the console.

Number values

Let's start with numbers.

Remember, in the end, computers are simply blazing fast calculators

2 + 2;

Some languages distinguish between integer and real numbers (real numbers are often called floats, due to the floating point technique used to represent them).

JavaScript does not make that distinction, everything is a real number (or simply, a Number).

When explicitly using decimal values, you may experience issues with precision.

Try to work with integers as much as you can, especially when you compare quantities in conditions and loops (if something, while something, etc.) 

0.1 + 0.2;

String values

Strings are simply, an entry of text.

However, string is a well-established term in Computer Science that appears in almost every programming language nowadays. Therefore, we will be using the term string a lot.

We have three ways for describing strings in JavaScript: 

`Using backticks`;
"Using doublequotes";
'Or using singlequotes';

All of them describe a string value.

They have slight differences but nothing to worry about for now.

Use any convention you want.

Escaping characters

Some characters are difficult to write in a string, for one reason or another.

These characters are often escaped with a backslash \.

A tab is encoded as \t. A newline is encoded as \n. In some instances (e.g., in the terminal of your operating system), a space is escaped as \ (backslash followed by a space).

You will almost never need to escape a space in JavaScript, but you will need to escape tabs and newlines.

Array values

An array is a special variable that can hold more than one value at a time.

Arrays are written in brackets, separated by commas

let albums = ["Ratatat", "Classics", "LP3", "LP4"];

You can read or set an array element by referring to its index number.

albums[3];
albums[4] = "Magnifique";

You can read characteristics of the array by looking at its properties. For example, to know the current length of the array:

albums.length

Dictionaries and objects

Dictionary (Key/Value pair collection) is a data type that can be useful. They can contain many values.

var release = { albumName: "Classics", 
              artistName: "Ratatat", 
              year: 2006, 
              label: "XL Recordings" };

New attributes can be easily added

release['members'] = "Evan Mast, Mike Stroud"

or

release.members = "Evan Mast, Mike Stroud"

FunctionsActions that to perform a particular task

function addition (a, b) {
  return a + b;
}

Functions are executed when

  • are called from JavaScript code
  • triggered by an event
  • are self-invoked
x = addition(release.artistName, release.albumName)

JavaScript variables (Bindings) Variables are identifiers that we can use to store data values 

let x, length;
x = 5;
length = 6;

or

let x = 5;
let length = 6;

Arithmetics operations can be performed suing JavaScript variables, as in:

let y = 5;

JavaScript Events

JavaScript can be used to execute code when events are detected

These events are actions that happen to HTML elements. For example:

  • An HTML element finished loading
    • <element onload="myScript()">
  • An HTML input field was changed
    • <select onchange="myFunction()">
  • An HTML button was clicked
    • <button type="button" onclick=otherScript()>

The script tagYour code goes here.

  • Lines between script tags contain the JavaScript code, and the browser will interpret the code between them as JavaScript.
  • Scripts can be placed in any section of HTML code, however putting scripts at the bottom of the body section of a web page can reduce display time.
  • Separating HTML and JavaScript by putting all the code in one place, is always a good habit.
  • Similarly to CSS files, JavaScript can also be placed in external files. This is practical when reusing the same code.
  • JavaScript files have the file extension .js. To use an external script, put the name of the script file in the source (src) attribute of the script tag:
    • <script src="myScript.js"></script>
