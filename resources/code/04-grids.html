Grid layout

The CSS Grid Layout Module offers a grid-based layout system, with rows and columns, making it easier to design web pages without having to use floats and positioning.

CSS libraries and frameworks

It is also possible to load and use external CSS files that come bundled with framework for web design. These are useful for creating responsive sites in an easy way.

Some libraries are: Bootstrap, Pure CSS, and Bulma
