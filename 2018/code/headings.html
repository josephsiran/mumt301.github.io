Links

Links are found in nearly all web pages. Links allow users to click their way from page to page.
HTML links are defined with the <a> tag, as in

<a href="url">link element</a>

The link address is specified in the href attribute
If we change the url to:

http://www.music.mcgill.ca/~gabriel/index.html

A link for going to the homepage would be written in HTML as:

<a href='http://www.music.mcgill.ca/~gabriel'>home</a>

A link for going to an external page could be:

Here is the Eighteenpl.us SoundCloud webpage
